Modèle Chain Ladder

Algorithme Chain Ladder L’algorithme Chain Ladder repose sur les deux hypothèses suivantes :

  • (CL1)) les montants cumulatifs pour différentes années de survenance

  • (CL2)) il existe des facteurs de développement (ou facteurs multiplicatifs ou facteurs de déroulement) λ1, …, λJ − 1 tels que λj représente le développement de la période j à la période j + 1, ou encore de la colonne j à la colonne j + 1 dans un triangle de développement : Ci(j+1) = λjCij pour j = 1, …, J − 1 et i = 1, …, J

Les facteurs de développement λk sont estimés à l’aide des observations: $$\widehat{\lambda}_k = \frac{\sum\limits_{i=1}^{J-k}C_{i(k+1)}}{\sum \limits_{i=1}^{J-k}C_{ik}} pour k=1,...,J-1$$
A partir de ces estimateurs λ̂k, on peut estimer le cout total pour chacune des ann´ees de survenance et obtenir une estimation pour la réserve. ĈiJ = (λ̂J − i + 1×...×λ̂J − 1)Ci(J−i+1)

R̂i = ĈiJ − Ci(J−i+1) R̂ = ∑R̂i

Modèle Mack

Modèle de Mack Les hypothèses du modèle de Mack sont

  • (MC1) indépendance par ligne : (Cij)j = 1, ..., J et (Ci′j)j = 1, ..., J sont indépendants pour i ≠ i′ ;

  • (MC2) on a la relation E[Cij|Ci1,...,Ci(j−1)] = λj − 1Ci(j−1), j = 2, ..., J

Estimateur du cout à l’ultime

Proposition 1 Sous les hypothèses (MC1) et (MC2), on a E[CiJ|𝒟J] = E[CiJ|Ci(J−i+1)] = λJ − 1λJ − 2...λJ − i + 1Ci(J−i+1), 1 ≤ i ≤ J

Proposition 2 Sous les hypothèses (MC1) et (MC2),

  1. Sachant ℬj , l’estimateur λ̂j est sans biais pour λj; et

  2. l’estimateur λ̂j est sans biais pour λj

Proposition 3 Sous les hypothèses (MC1) et (MC2),

  1. Sachant Ci(J−i+1), ĈiJ est un estimateur sans biais pour E[CiJ|Ci(J−i+1)]

  2. ĈiJ est sans biais pour E[CiJ]

  3. les estimateurs λ̂1, ..., λ̂j sont non correlés

Estimateur pour l’erreur de prédiction

Une méthode stochastique a l’avantage de permettre l’estimation de l’erreur de prédiction.Pour le modèle de Mack, il faut modifier (compléter) la seconde hypothèse afin de pouvoir calculer l’erreur de prévision :

$$\widehat{\sigma}_{j^2} = \frac{1}{J-j-1} \sum \limits_{i=1}^{J-j}C_{ij} \left( \frac{C_{i(j+1)}}{C_{ij}} - \widehat{\lambda}_j \right)^2$$

$$\widehat{\sigma}_{J-1}^2 = min \left\{ \frac{\widehat{\sigma}_{J-2}^4}{\widehat{\sigma}_{J-3}^2}, min \left\{ \widehat{\sigma}_{J-3}^2, \widehat{\sigma}_{J-2}^2 \right\} \right\}$$

Proposition 4 Sous les hypothèses (MC1) et (MC2), on a

  1. sachant ℬj,l’estimateur σ̂j2 ; est sans biais

  2. l’estimateur σ̂j2 est sans biais pour σj2

Erreur quadratique moyenne de prédiction Pour une variable aléatoire X, on suppose que X̂ est un estimateur ℬ-mesurable pour E[X|𝒟] et un prédicteur 𝒟-mesurable pour X. On définit l’erreur quadratique moyenne conditionnelle de prédiction par

$$\begin{split} MSEP_{X|\mathcal{D}}(\widehat{X}) & = E \left[ \left( \widehat{X} - X \right)^2 | \mathcal{D} \right] \\ & = \underbrace{Var\left[X|\mathcal{D} \right]}_{\mbox{erreur stochastique}} + \underbrace{ \left( \widehat{X} - E \left[X|\mathcal{D} \right] \right)^2}_{\mbox{erreur d'estimation}} \end{split}$$

Erreur stochastique

Pour une année de survenance i > 1, on a: $$Var[C_{iJ}| \mathcal{D}_J] = \sigma_{J-1}^2C_{i(J-i+1)} = \prod_{m=J-i+1}^{J-2} \lambda_m + \lambda_{J-1}^2 Var\left[ C_{i(J-1)}|C_{i(J-i+1)} \right]$$ $$Var[C_{iJ}| C_{i(J-i+1)} ] = \left( E\left[ C_{iJ}|C_{i(J-i+1)} \right] \right)^2 \sum \limits_{j=J-i+1}^{J-1} \frac{\sigma_j^2/\lambda_j^2}{E[C_{ij}|C_{i(J-i+1)}]}$$

Erreur d’estimation

$$\widehat{MSEP}_{C_{ij}|\mathcal{D}_J}^B (\widehat{C}_{iJ}) \approx C_{iJ}^2 \sum \limits_{i=J-i+1}^{J-1} \frac{\sigma_j^2}{\lambda_j^2} \left( \frac{1}{C_{ij}} +\frac{1}{\sum_{i=1}^{J-j}C_{ij}} \right)$$

Proposition 5 L’estimateur de l’erreur quadratique moyenne conditionnelle de prédiction pour l’ensemble des années de survenance est donné par $$\begin{split} \widehat{MSEP}_{\sum_i C_{iJ}|\mathcal{D}_J}^A \left( \sum_{i=1}^{J}\widehat{C}_{iJ} \right) = \sum_{i=1}^J \widehat{MSEP}_{C_{iJ}|\mathcal{D}_J}^A(\widehat{C}_{iJ}) \\ + 2\sum_{1 \leq i < k \leq J} C_{i(J-i+1)}\widehat{C}_{k(J-i+1)} \left( \prod \limits_{j=J-i+1}^{J-1} \left( \widehat{\lambda}_j^2 + \frac{\sigma_j^2}{\sum_{i=1}^{J-j} \widehat{C}_{ij}} \right) \prod \limits_{j=J-i+1}^{J-1} \widehat{\lambda}_j^2 \right) \end{split}$$
